Radovan, In your home directory, you can create a script to both add the missing folder/file and start Oracle:

#!/bin/bash
sudo mkdir /var/lock/subsys
sudo touch /var/lock/subsys/listener

sudo service oracle-xe start
---
Then change the permissions on the file and run it.﻿
